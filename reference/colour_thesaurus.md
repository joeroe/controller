# Colour thesaurus

A dataset of colour names based on Ingrid Sundberg's "colour thesaurus".
Contains 240 names for shades of 12 colours, formatted for use as a
thesaurus in
[`control()`](https://controller.joeroe.io/reference/control.md).

## Usage

``` r
colour_thesaurus
```

## Format

A data frame with 240 rows and 2 variables:

- colour:

  preferred colour name: `"white"`, `"tan"`, `"yellow"`, `"orange"`,
  `"red"`, `"pink"`, `"purple"`, `"blue"`, `"green"`, `"brown"`,
  `"grey"`, or `"black"`

- shade:

  variant names for shades of `colour`
